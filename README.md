# 各版本台灣共和國憲法草案

![](taiwan-flag.jpg)

此處整理台灣共和國憲法草案之相關版本，促進台灣獨立/台澎終止代管自決建國理念之討論之討論。若有其他版本未收錄，敬請來信告知。

有關台灣獨立建國運動，請見[維基百科條目](https://zh.wikipedia.org/zh-tw/%E5%8F%B0%E7%81%A3%E7%8D%A8%E7%AB%8B%E9%81%8B%E5%8B%95)。
